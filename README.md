# Finartz Project
This is a project which you can add,update,delete and get some informations from Mongodb database with Spring MVC.

### 1. Technologies Used

* Spring
* Jackson
* jQuery
* Boostrap
* Ajax
* Maven

### 2. To Get Code ###

clone https://github.com/suleymangunduzz/FinartzProject.git


### 3. To Run This Project Locally

* You should have the MongoDb on your machine and you can use the Eclipse IDE for to run this project.
* You can configure the database informations from application.properties file.
* This is a Spring Boot Application so you can run the java class which named Application.class by clicking the Run Application option.
* After that open the https://localhost:8080 link in your browser to see what does this project excatly do.
